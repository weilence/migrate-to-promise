## Migrate `toPromise`

This is a tool for migrating the toPromise function in RxJS.

## How to use

```shell
pnpm install
pnpm start /path/to/project/source/code/**/*.ts
```

`/path/to/project/source/code` should not contains `node_modules`, it causes very slow.
